1 Aufgaben

1.1 Beschreibung

Es soll eine HTML Seite entsprechend der beiliegenden Vorlagen „vorlage_desktop.png“ und „vorlage_mobil.png“ erstellt werden

Folgende Hilfsmittel dürfen eingesetzt werden

  • jQuery
  • Bootstrap (o.ä.)
  • Fontawesome
  • Less / SCSS (mit Hilfe eines Online Kompiliers in CSS umwandeln)

Die benötigten Bilder sind im Ordner abgelegt.

1.2 Allgemeine Kriterien

  • Die nachgebildete Seite sollte dem Original im Wesentlichen entsprechen, muss aber natürlich nicht pixelgenau nachgebaut werden
  • Das Aussehen soll für die Auflösungen Iphone 5 (320x568), IPad Hochformat (768x1024) und Desktop (Breiten ab 1200px) geprüft und angepasst werden
  • Hauptaugenmerk darf auf einem beliebigen Browser der eigenen Wahl liegen, jedoch sollten abschließend auch die Browser Chrome, Firefox und Edge kurz geprüft und evtl leicht angepasst werden
  • Die HTML-Struktur sollte gemäß üblichen SEO-Kriteriuen aufgebaut sein

1.3 Details zum Header

Der Header soll als Slider umgesetzt werden. Zusätzlich zur Vorlage soll es Navigationspfeile geben.

1.4 Details zur Navigation

Bei Mouse Hover soll sich die Schriftfarbe zu blau ändern und eine blaue Linie erscheinen. Des Weiteren soll die Navigation etwas mehr Abstand nach oben erhalten (siehe vorlage_navigation_hover.png).

1.5 Details Klapper

Die Elemente „Ihr Lächeln ist unser schönstes Kompliment“ und „Vereinbaren Sie einen Termin mit uns.“ sollen sich bei Klick en und einen Textabschnitt zeigen. Hierfür bitte nicht auf Bootstrap oder andere Bibliotheken zurückgreifen, sondern eine eigene JavaScript (oder JQuery) Lösung erstellen.

1.6 Details Google Map

Es soll ein Google Maps Snippet eingebunden werden mit markierten Standort und einem Feld mit Adressinformationen. Optional kann der markierende Pin kundenspezifisch angepasst werden.
